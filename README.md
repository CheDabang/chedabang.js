###  使用说明：
1、折线图：
```
 new Line(data,id); 语法的格式
```
**具体实例**
```
<script type="text/javascript">
	var data = {
		0: 260,
		1: 200,
		2: 168,
		3: 450,
		4: 360,
		5: 320,
		6: 237,
		7: 189,
		8: 257,
		9: 389,
		10: 289,
		11: 236
	}
	var line = new Line(data, "#draw");
</script>
```

这里首先按照此格式设置一个折线图的数据点，之后实例化Line构造函数。传入data数据，和你的canvasID值。之后就能够生成一个折线图。
